# LSTM-Stock-Price-Prediction
Research on Stock Price Prediction Method Based on Optimized LSTM Neural Network
基于优化LSTM神经网络的股票价格预测方法的研究
股票价格预测是金融时间序列分析中的核心问题。针对股票市场数据具有非线性、高噪声和时变性的特点，传统统计学模型难以捕捉其深层规律，本项目提出了一种基于技术指标特征增强与超参数优化的长短期记忆网络（LSTM）预测模型。首先，对2021年至2025年的黄金股票历史交易数据进行清洗与归一化处理，并计算移动平均线（MA）、相对强弱指数（RSI）、布林带（Bollinger Bands）及平滑异同移动平均线（MACD）等多维技术指标作为输入特征；其次，构建LSTM神经网络模型，利用其门控机制捕捉时间序列的长短期依赖关系；进而，采用网格搜索算法（Grid Search）对模型的神经元个数、Dropout率、优化器及批次大小等超参数进行寻优。实验结果表明，优化后的模型在测试集上的均方误差（MSE）从0.3229降低至0.1311，降幅达59.4%，均方根误差（RMSE）和平均绝对误差（MAE）亦显著下降。该方法有效解决了传统模型在复杂市场环境下预测精度低、滞后性强的问题，具有较高的应用价值。


环境配置：
本实验在 Windows 10 操作系统下进行，基于 Python 编程语言和深度学习框架实现。主要依赖库及其版本如下：
(1)编程语言：Python 3.9
(2)深度学习框架：TensorFlow 2.10.0 / Keras 2.10.0 (用于构建 LSTM 及全连接层)
(3)Pandas 1.5.3 (数据清洗与时间序列处理)
(4)NumPy 1.23.5 (数值计算)
(5)Scikit-learn 1.2.2 (提供 Grid Search 算法及 Min-Max 归一化工具) 
(6)绘图工具：Matplotlib 3.7.1 (用于绘制 K 线图及预测结果对比图)
